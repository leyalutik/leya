Общая структура данных

Исходники (source files)
	main.cpp 	 - запуск программы
	lessons.cpp - набор упражнений для чтения
	bricks.cpp   - элементарные фукнции, которые используют функции из lessons.cpp
	parse_file.cpp -   библиотека для считывания текстовых файлов

Заголовочные файлы (header files)
	lessons.h - набор упражнений для чтения
	bricks.h   - элементарные фукнции, которые используют функции из lessons.cpp
	parse_file.h -   библиотека для считывания текстовых файлов

Вспомогательные текстовые файлы
	russian_words.txt	 	-  набор из 1000 русских слов
	english_words.txt		- набор из 1000 английских слов
	english_theme_cards.txt		- набор из тематических карточек на английском, где в одной карточке 5-7 слов одного смысла.
					(это реализовано, через идущие подряд строки. Каждая карточка 
						разделена пробелом)
	russian_theme_cards.txt	- набор из тематических карточек на русском, где в одной карточке 5-7 слов одного смысла.
					(это реализовано, через идущие подряд строки. Каждая карточка 
						разделена пробелом)
